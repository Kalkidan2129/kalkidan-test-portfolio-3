
# Revenue Forecasting and Performance Analysis of Vodafone Qatar (2021–2023)

## Project Summary

This project transforms Vodafone Qatar’s 2021‑2023 telecom data into a unified, ML‑ready dataset using NexusMax, Python, and Microsoft Fabric. The enriched data powers a Power BI dashboard that visualizes revenue, churn, ARPU, and network performance, enabling finance and strategy teams to forecast quarterly revenue and identify churn risks with greater precision.

---

## Business Problem

Telecom revenue performance varies across regions and segments, yet finance and strategy teams lack a consolidated view to understand growth drivers and revenue risks. Fluctuations in churn and subscriber behavior create uncertainty in forecasting and budget planning.

---

## Objective

- Unify and enrich raw Vodafone Qatar telecom data into a single, ML‑ready dataset.
- Develop a Power BI dashboard that visualizes key KPIs such as revenue, churn, ARPU, and network availability.
- Enable accurate quarterly revenue forecasting and churn reduction insights for strategic decision‑making.

---

## Tools & Technologies

- NexusMax
- Python (Pandas)
- Microsoft Fabric
- Power BI
- SQL
- Data Engineering
- Data Visualization
- Machine Learning

---

## Project Workflow

- Collect and curate raw telecom data from Kaggle and Vodafone investor presentations.
- Clean, standardize, and enrich the dataset with NexusMax and Python, creating a centralized date table and validated KPIs.
- Load the prepared data into Microsoft Fabric for storage and ML readiness.
- Build a Power BI dashboard to visualize revenue, churn, ARPU, and network performance trends.
- Apply supervised regression techniques to forecast quarterly revenue and identify churn drivers.

---

## Key Insights

- Total revenue grew 14% QoQ to $108K, with Doha contributing the largest share supported by 72M subscribers.
- High‑value mobile prepaid churn remains above 2.5%, rising 0.3% QoQ, threatening revenue stability.
- Network availability is exceptionally high at 99.95%, providing a strong foundation for retention efforts.
- Churn in Mobile Postpaid and Fixed Broadband segments ranges 2.41%–2.74%, exceeding the overall churn rate and causing an estimated $8–12K annual revenue loss.
